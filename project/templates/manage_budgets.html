Aide

Saisie et suivi des codes budgétaires

Cette page permet d'attribuer, de modifier et de suivre rapidement les identifiants budgétaires des fiches projets.

1. Attribution et saisie

  • Sélection rapide : Cliquer dans le champ de saisie pour faire apparaître la liste des codes budgétaires déjà existants. Choisir un code (faire défiler la liste si nécessaire).
  • Nouveau code : Taper directement un nouveau code inédit. Il s'ajoutera automatiquement aux suggestions après enregistrement.
  • Suppression : Effacer complètement le texte du champ et valider pour retirer l'attribution budgétaire d'un projet.

2. Sauvegarde automatique

L'enregistrement s'effectue automatiquement en appuyant sur Entrée ou en cliquant en dehors du champ. La colonne État indique l'état en temps réel :

  • Enregistré : le code actuel est synchronisé avec la base de données.
  • Non enregistré : la modification du champ est en cours.
  • Sauvegarde en cours : le serveur traite la modification.
  • Succès : la dernière modification a bien été enregistrée.
  • Erreur : une contrainte n'a pas été respectée ou un problème est survenu.

3. Règles et contraintes

Un code budgétaire doit comporter entre 3 et 50 caractères (ou être totalement vide). Toute modification non-conforme n'est pas prise en compte.

4. Navigation et productivité

  • Touche Entrée : Appuyer sur Entrée après avoir saisi un code pour valider l'enregistrement et passer automatiquement à la ligne suivante.
  • Détails du projet : Cliquer directement sur le titre d'un projet pour ouvrir un aperçu détaillé de ses demandes financières.
  • Fiche complète : Cliquer sur le badge de l'identifiant projet (Id) pour ouvrir la page du projet.
