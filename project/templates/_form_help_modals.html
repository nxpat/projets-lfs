Date et heure

  • Indiquer la date à laquelle se déroule le projet.
  • Pour un projet se déroulant sur une période, indiquer les dates de début et de fin du projet.
  • Il est nécessaire d'indiquer l'heure de début et l'heure de fin pour une sortie scolaire ou un projet pouvant impacter les autres cours de l'emploi du temps habituel des élèves.
  • Pour un projet se déroulant sur plus d'une année scolaire, créer une fiche projet par année scolaire. On pourra utiliser le bouton .
  • Un projet récurrent sera dupliqué chaque année.

Aide fiche projet

Équipe pédagogique

Sélectionner les membres de l'équipe pédagogique participant au projet.

  • Pour une sortie scolaire, sélectionner aussi les accompagnateurs de la sortie. Les stagiaires seront indiqués dans le champ Encadrement (stagiaires et personnes extérieures au LFS).
  • Pour sélectionner plusieurs personnes, appuyer sur la touche ou sur macOS et cliquer sur le nom de la personne (faire défiler la liste si nécessaire). Répéter cette opération autant de fois que nécessaire.
  • Pour sélectionner une plage de noms, cliquer sur le premier nom de la sélection, puis appuyer sur la touche et cliquer sur le dernier nom de la sélection (faire défiler la liste si nécessaire).

La fiche projet ne permet pas d'associer un rôle aux membres de l'équipe. Vous pouvez cependant, si vous le souhaitez, décrire le rôle de chacun dans la partie Description du projet.

Liste des élèves

Si la participation au projet est optionnelle, préciser la liste des élèves avant la demande validation.

Indiquer un élève par ligne avec la classe, le nom et le prénom séparés par une virgule, deux espaces ou une tabulation. La classe doit être indiquée dans la première colonne. La classe peut être omise si une seule classe est concernée.

Il est possible de copier / coller la liste depuis votre tableur préféré (Google Sheets, LibreOffice Calc, MS Excel, etc.).
Si vous copiez la liste depuis Pronote, vous devez ajouter une virgule ou au moins deux espaces entre le nom et le prénom (en effet, Pronote met le nom et le prénom dans la même colonne).

Le format accepté pour les classes (avec ou sans espace avant le nom ou le numéro de la classe, avec ou sans majuscules) :

Terminale, Tale, Tle, 1ère, 1ere, 1re, 1e, 2e, 2de, 2nde, 3ème, 3eme, 3e, cm2, cp, gs/ms.

Exemples :

Une tabulation entre chaque colonne, comme lorsqu'on importe la liste d'un tableur

Une virgule entre chaque colonne

Une virgule ou une tabulation entre chaque colonne

Deux espaces entre chaque colonne

Une seul espace entre les colonnes

La 2de C n'existe pas au LFS !
